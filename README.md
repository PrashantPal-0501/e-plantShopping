# Paradise Nursery — E-Plant Shopping App

A React + Redux Toolkit e-commerce app for a houseplant nursery, **Paradise
Nursery — Where Green Meets Serenity**. Built with Vite, React, and Redux
Toolkit for state management.

## Features

- **Landing page** — background image, company name, an "About Us" paragraph,
  and a "Get Started" button that leads to the product listing.
- **Product listing page** — nine houseplants across three categories (Air
  Purifying Plants, Aromatic Fragrant Plants, Succulents), each with a
  thumbnail, name, price, and an "Add to Cart" button that disables once the
  plant has been added.
- **Shared header** — appears on both the product listing and cart pages,
  with navigation links (Home / Plants / Cart) and a cart icon that updates
  dynamically with the total item count.
- **Shopping cart page** — shows every plant type in the cart with its
  thumbnail, name, unit price, running total, quantity +/- controls, and a
  delete button, plus the overall cart total, a "Continue Shopping" button,
  and a "Checkout" button (shows a "Coming Soon" message).

## Tech stack

- React 19 + Vite
- Redux Toolkit (`@reduxjs/toolkit`) + React-Redux for cart state
